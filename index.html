About XComponents

XComponents is a set of AngularJS directives that allow you to quickly create a simple app. It is build on top of the Bootcards UI framework to get a near-native look on iOS and Android.

Getting started

  • Download the latest release from this repository - see link above (or clone it from git clone https://github.com/bootcards/xcomponents).
  • Install it using Bower: bower install xcomponents
