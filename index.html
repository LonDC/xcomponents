About XComponents

XComponents is a set of AngularJS directives that allow you to quickly create a simple app. It is build on top of the Bootcards UI framework to get a near-native look on iOS and Android.

Getting started

To get started with XComponents:

  • Download the latest release from this repository - see link above (or clone it from git clone https://github.com/bootcards/xcomponents).
  • Install it using Bower: bower install xcomponents

Check out the demo app or start a new app yourself. Let's get started building a simple app that uses the XComponents and talks to a remote REST API build using Strongloop Loopback.

1: Required software

Install Node and Bower. Then install Strongloop Loopback in Node: npm install -g strongloop.

2: Setup the app

Create a new folder for you app. Using a terminal, go into the folder and enter slc loopback. This will setup the basic Loopback application.

3

Create a file called .bowerrc in the new folder and set its contents to: 

      {
  "directory": "client/bower_components/"
}
This will tell bower to install all dependencies into the client/bower_components folder.

4: Setup up the server

Create a data model: let's create a list of cars. Enter: slc loopback:model. Accept all defaults until you get to "Properties". Create the following properties: 'make', 'type' and 'color'. Leave the property name empty to stop creating properties.

Delete the file server/boot/root.js. We don't need it. Open 'server/middleware.json' and replace the file property with: "files": { "loopback#static": { "params": "$!../client" } },

5: Install XComponents

Run bower install xcomponents --save to download XComponents and all its dependencies.

6: Basic page

Create a file called index.html in the /client folder and set it contents as follows to create a simple form/view page: 
<!doctype html>
<html>

<head>
  <title>XComponents</title>
  <meta name="viewport" content="width=device-width, initial-scale=1">

  <!--set the hidden class here, so everything is immediately hidden-->
  <style type="text/css">
    .hidden {
      display:none;
    }
  </style>
  
  <!-- polyfill for html imports -->
  <script type="text/javascript" src="/bower_components/webcomponentsjs/HTMLImports.min.js"></script>
  <link rel="import" href="/bower_components/xcomponents/dist/includes/includes.html">
 
  <script src="cars.js"></script> 
  
</head>

<body ng-controller="xcController" class="hidden">

  <xc-header title="View my cars"></xc-header>

  <xc-list 
    title="Cars"
    list-width="5" 
    allow-add="true"
    order-by="make"
    order-reversed="false"
    url="/api/Cars/:id"
    summary-field="make"
    details-field="model"
    image-field="thumbnailUrl"
    image-placeholder-icon="car"
    autoload-first="true"
    type="categorised"
    group-by="make"
    infinite-scroll="true">

    <xc-form 
      url="/api/Cars/:id" 
      default-text="Select a car from the list"
      thumbnail-field="thumbnailUrl"
      thumbnail-show-with="name"></xc-form>

  </xc-list>

  <xc-footer></xc-footer>

</body></html>

7: JavaScript config file

Create a file called cars.js in the /client folder and set it contents as follows: 
 var xcontrols = {

  appVersion : '0.1',

  menuOptions : [

    { label : 'Languages', isSecondary : true, icon : 'fa-globe' ,
      menuOptions : [
        { label : 'Dutch', url : 'contacts-accordion.html'},
        { label : 'English', url : 'contacts-accordion.html'}
      ]
    },
    { label : 'Help', isSecondary : true, icon : 'fa-question-circle'},
    
    { label : 'Dashboard', url : 'index.html', icon : 'fa-dashboard' },
    { label : 'Lists', url : 'lists.html', icon : 'fa-list',
      menuOptions : [
        { label : 'Cars', url : 'index.html', icon : 'fa-list-alt' }
      ]
    },
    
  ],

  footerOptions : [
    { label : 'Cars', url : '/index.html', icon : 'fa-dashboard'}
  ], 

  modelName : 'Car', 

  fields : [
    { label : 'Make' , field: 'make', required: true},
    { label : 'Model' , field: 'model' },
    { label : 'Color' , field: 'color' },
  ]

}

8: Start the server

Start the local server with slc run and view the result in at http://localhost:3000. The app has a different look and feel on iOS, Android and Desktop and auto-adapts to the device size, including changing its behaviour. On tablets you get independent scrolling columns. On a smartphone a simple list/detail view with a back button.
